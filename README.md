# Task Management Application

A modern full-stack Task Management Application developed using React.js, Spring Boot, and MySQL. The application allows users to securely create, update, delete, and track tasks with authentication and authorization.

## Features
- User Registration & Login (JWT Authentication)
- User Authentication & Authorization
- Create, Read, Update, Delete (CRUD) Tasks
- Task Priority (Low, Medium, High)
- Task Status (Pending, In Progress, Completed)
- Due Date Management
- Search and Filter Tasks
- Responsive Design for Desktop & Mobile
- RESTful APIs with Spring Boot
- MySQL Database Integration
- Optional Real-Time Updates using WebSockets

## Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Spring Boot, Spring Security, JWT
- Database: MySQL
- Build Tool: Maven
- Version Control: Git & GitHub

This project is designed to demonstrate full-stack development skills, secure authentication, API integration, and database management.
